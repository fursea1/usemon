Follow these steps to add another service with associated methods:

  1. Add the new service in UsemonServiceProvider.java
  2. Create the new service interface
  3. Add methods to your new service
  4. Add instantiation of the new service to UsemonServiceLocator
  5. Create an implemenation of the new service in the service layer. Annotate all injection points with @Inject
  6. Add the creation of the new service to UsemonServiceProviderImpl
  7. Add bindings required into DefaultModuleBinding
